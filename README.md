# PAU-CSC 212 - OpenGL n-sided polygon

A C++ OpenGL project using GLFW. It renders a purple transparent dodecahedron with a solid outline and a simple physics-inspired animation.

## Animation

The dodecahedron:

1. starts on the floor
2. levitates upward
3. rotates in the air
4. grows larger
5. falls under gravity with a small bounce

## Controls

- `Space` - pause / resume
- `Esc` - close the window

## Build

This project was made with Visual Studio 2022, CMake, OpenGL, and GLFW.

To build it, open the folder in Visual Studio 2022 and run the CMake target.
